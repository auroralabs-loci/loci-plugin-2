## Preflight: osal_mem_free

Call graph:  OK — osal_mem_free is a standalone memory-management routine; the fix touches only internal arithmetic within this function, no new callees are introduced.

Arithmetic:  ⚠ RISK — The core issue is an unsigned subtraction hazard. The expression computing free size is approximately:

    free_size = block->reported_size - BLOCK_HEADER_SIZE;

Both operands are unsigned (size_t or equivalent). If block->reported_size is corrupted, zero, or smaller than BLOCK_HEADER_SIZE, the subtraction wraps to a value near SIZE_MAX. Any subsequent use of free_size — memset, coalescing loop bound, sanity check — will operate on that huge value, causing out-of-bounds writes or infinite loops.

The fix must add a guard:
    if (block->reported_size < BLOCK_HEADER_SIZE) { /* handle error */ }
before the subtraction so the wrapped value is never produced.

Resources:   OK — no new resource acquisition or release is introduced by this fix; the guard is pure control-flow.

Execution fit: ADJUST PLAN
→ Add an explicit underflow guard before the subtraction; decide on the error path (assert, return, or mark the block as corrupt) before writing any code.

## Adjusted Plan

### Step 1 — Read the current implementation
Read osal_mem_free in osal_memory.c to find the exact subtraction expression and identify which error path is appropriate (assert, early return, or corruption flag).

### Step 2 — Choose the error path
Options:
- `OSAL_MEM_ASSERT(block->reported_size >= BLOCK_HEADER_SIZE)` — crashes on corrupt heap, safest for embedded debug
- `return` early — silently skips the free, avoids crash but leaks
- Set a corruption flag and return — allows diagnostics

Recommend the assert approach for debug builds, with a silent return for release.

### Step 3 — Add the guard
```c
if (block->reported_size < BLOCK_HEADER_SIZE) {
    OSAL_MEM_ASSERT(0);   /* heap corruption detected */
    return;
}
free_size = block->reported_size - BLOCK_HEADER_SIZE;
```

### Step 4 — Verify all other unsigned subtractions in the function
Check for any other `x - y` where both are unsigned and y could exceed x.
